# ❓ FAQ - Häufig gestellte Fragen

Die häufigsten Fragen und Antworten zur QuizMaster Plattform.

## 🚀 **Allgemeine Fragen**

### Was ist QuizMaster?
QuizMaster ist eine webbasierte Quiz-Plattform, die es ermöglicht, interaktive Quiz zu erstellen, zu verwalten und in Echtzeit durchzuführen. Die Plattform ist vollständig in deutscher Sprache verfügbar und erfordert keine Registrierung oder Installation.

### Ist QuizMaster kostenlos?
Ja, QuizMaster ist vollständig kostenlos und open source. Sie können die Plattform ohne Einschränkungen nutzen.

### Brauche ich einen Account?
Nein, Sie benötigen keinen Account. Die Plattform funktioniert ohne Registrierung. Quiz werden temporär verwaltet und Einstellungen lokal im Browser gespeichert.

### Auf welchen Geräten funktioniert QuizMaster?
QuizMaster funktioniert auf allen modernen Geräten:
- **Desktop/Laptop**: Windows, Mac, Linux
- **Tablets**: iPad, Android-Tablets
- **Smartphones**: iPhone, Android-Phones
- **Browser**: Chrome, Firefox, Safari, Edge

## 🎨 **Theme und Design**

### Wie ändere ich zwischen Light und Dark Mode?
Klicken Sie auf das Theme-Symbol (🌙 oder ☀️) in der Navigation. Die Plattform erkennt auch automatisch Ihre Systemeinstellungen.

### Wird meine Theme-Einstellung gespeichert?
Ja, Ihre Theme-Präferenz wird lokal in Ihrem Browser gespeichert und bei jedem Besuch wiederhergestellt.

### Warum sehe ich manchmal das falsche Theme?
Das kann passieren, wenn:
- Sie einen anderen Browser verwenden
- Der Browser-Cache geleert wurde
- Sie im Inkognito-Modus surfen

## 🚪 **Quiz beitreten**

### Wie trete ich einem Quiz bei?
1. Gehen Sie zur "Quiz beitreten" Seite
2. Geben Sie Ihren Namen ein
3. Geben Sie den Quiz-Namen ein (erhalten von Ihrem Quiz-Moderator)
4. Klicken Sie auf "Quiz beitreten"

### Was ist ein Quiz-Name?
Der Quiz-Name ist eine eindeutige Bezeichnung für ein Quiz, die vom Administrator vergeben wird. Beispiele: "mathe-test-2024", "geschichte-europa"

### Ich kann nicht beitreten - "Quiz nicht gefunden"
Überprüfen Sie:
- ✅ Quiz-Name korrekt geschrieben (keine Tippfehler)
- ✅ Quiz ist noch aktiv
- ✅ Moderator hat das Quiz gestartet
- ✅ Ihre Internetverbindung funktioniert

### Kann ich meinen Namen später ändern?
Derzeit nicht während einer aktiven Session. Sie müssten das Quiz verlassen und mit einem neuen Namen erneut beitreten.

## 🔧 **Quiz erstellen (Admin)**

### Wie erstelle ich ein Quiz?
1. Gehen Sie zum Admin Panel
2. Klicken Sie auf "Quiz erstellen"
3. Geben Sie einen Quiz-Namen ein
4. Fügen Sie Fragen und Antworten hinzu
5. Konfigurieren Sie die Einstellungen
6. Speichern Sie das Quiz

### Wie viele Fragen kann ich hinzufügen?
Technisch gibt es keine Begrenzung, aber für die beste Benutzererfahrung empfehlen wir:
- **Kurze Quiz**: 5-10 Fragen
- **Mittel**: 10-20 Fragen
- **Lang**: 20-50 Fragen

### Kann ich Bilder in Fragen verwenden?
Derzeit unterstützt die Plattform nur Text-basierte Fragen. Bild-Support ist für zukünftige Versionen geplant.

### Wie lösche ich ein Quiz?
Im Admin Panel unter "Meine Quiz" finden Sie eine Lösch-Option für jedes Quiz.

## 🔴 **Live-Sessions**

### Wie starte ich eine Live-Session?
1. Erstellen Sie ein Quiz im Admin Panel
2. Fügen Sie Teilnehmer hinzu
3. Wechseln Sie zum Live Control Panel
4. Klicken Sie auf "Session starten"

### Wie viele Teilnehmer können gleichzeitig teilnehmen?
Die Plattform unterstützt theoretisch unbegrenzt viele Teilnehmer. Für optimale Performance empfehlen wir:
- **Optimal**: 10-30 Teilnehmer
- **Gut**: 30-100 Teilnehmer
- **Bei mehr Teilnehmern**: Performance kann variieren

### Was passiert wenn jemand die Verbindung verliert?
- Die Person kann sich erneut anmelden
- Der Fortschritt geht verloren
- Bei wichtigen Quiz sollten Sie eine Backup-Strategie haben

## 🔧 **Technische Probleme**

### Die Seite lädt nicht oder sehr langsam
**Lösungsschritte**:
1. Überprüfen Sie Ihre Internetverbindung
2. Aktualisieren Sie die Seite (F5 oder Ctrl+R)
3. Leeren Sie den Browser-Cache
4. Versuchen Sie einen anderen Browser
5. Prüfen Sie ob andere Websites funktionieren

### "Verbindung zum Server fehlgeschlagen"
**Mögliche Ursachen**:
- Temporäre Netzwerkprobleme
- Server-Wartung
- Firewall-Blockierung

**Lösungen**:
- Warten Sie ein paar Minuten und versuchen es erneut
- Überprüfen Sie Ihre Firewall-Einstellungen
- Versuchen Sie ein anderes Netzwerk

### JavaScript-Fehler oder "Seite funktioniert nicht richtig"
**Lösungsschritte**:
1. Aktivieren Sie JavaScript in Ihrem Browser
2. Deaktivieren Sie Browser-Erweiterungen temporär
3. Versuchen Sie den Inkognito-/Private-Modus
4. Aktualisieren Sie Ihren Browser

### Mobile Probleme
**Häufige mobile Probleme**:
- Seite zu klein dargestellt → Zoomen Sie hinein
- Buttons schwer zu treffen → Verwenden Sie Portrait-Modus
- Langsame Performance → Schließen Sie andere Apps

## 📱 **Browser-spezifische Fragen**

### Welcher Browser ist am besten?
**Empfohlene Browser** (in dieser Reihenfolge):
1. **Chrome** - Beste Performance und Features
2. **Firefox** - Gute Alternative, alle Features

## 💾 **Daten und Speicherung**

### Werden meine Daten gespeichert?
- **Theme-Einstellungen**: Ja, lokal im Browser
- **Quiz-Daten**: Temporär während der Session
- **Persönliche Daten**: Nein, werden nicht dauerhaft gespeichert

### Kann ich Quiz-Ergebnisse exportieren?
Derzeit nur begrenzt möglich. Diese Funktion ist für zukünftige Versionen geplant.

### Was passiert wenn ich den Browser schließe?
- Laufende Quiz-Sessions werden unterbrochen
- Theme-Einstellungen bleiben erhalten
- Sie müssen sich erneut in aktive Quiz einloggen

## 🔒 **Sicherheit und Datenschutz**

### Ist QuizMaster sicher?
Ja, QuizMaster verwendet:
- HTTPS-Verschlüsselung
- Minimale Datensammlung
- Keine permanente Speicherung persönlicher Daten
- Open-source Code für Transparenz

### Welche Daten werden gesammelt?
- **Temporär**: Quiz-Namen, Teilnehmernamen während Sessions
- **Lokal**: Theme-Einstellungen im Browser
- **Nicht gesammelt**: E-Mail, persönliche Informationen, IP-Adressen

### DSGVO-Konformität
QuizMaster ist DSGVO-konform da:
- Keine persönlichen Daten permanent gespeichert werden
- Nutzer volle Kontrolle über ihre Daten haben
- Keine Tracking-Cookies verwendet werden
